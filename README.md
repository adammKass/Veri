# Virignon - Full-Screen Vertical Slider Prototype

A responsive UI prototype featuring a smooth **parallax effect**.  
Built with **React**, **Tailwind CSS**, and **Motion**.

---

## Features

### Full-Screen Horizontal Slider

- Smooth animated transitions
- Parallax background movement

### Technology

- **React** – component-based UI
- **Tailwind CSS** – utility-first styling
- **GSAP** – animation engine for gestures and parallax

## Installation

```bash
# Clone the repository
git clone https://github.com/adammKass/Veri

# Install dependencies
npm install

# Start the development server
npm run dev
```
